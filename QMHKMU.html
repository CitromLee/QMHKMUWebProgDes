GitHub Parancsok

Helyi tároló (local)

  • git init - Új git tároló létrehozása.
  • git add . - Minden fájl hozzáadása a következő commithoz.
  • git commit -m "Commit üzenet" - Változások elmentése a helyi tárolóban.
  • git checkout <branch_név> - Átváltás egy másik ágra.

Távoli tároló (remote)

  • git remote add origin <repository_url> - Távoli tároló hozzáadása.
  • git push origin master - A helyi változások feltöltése a távoli tárolóba.
  • git pull origin master - Frissítések letöltése a távoli tárolóból.
  • git fetch origin - A legfrissebb változások lekérése a távoli tárolóból anélkül, hogy azokat egyesítené a helyi tárolóval.
